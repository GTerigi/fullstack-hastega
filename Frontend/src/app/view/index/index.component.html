Prova Tecnica Hastega

Introduzione

Implementare una semplice applicazione web per gestire la propria libreria. Gli utenti possono aggiungere, modificare e rimuovere libri dalla propria libreria. Inoltre, ogni utente vuole tenere traccia del numero di letture per ogni libro.
